# Personal Finance Dashboard

## Project Description

The Personal Finance Dashboard is a simple interactive web application designed to help users track their income, expenses, and savings. The application allows users to add transactions, view financial summaries, and analyze their spending through visual charts and reports.

## Features

* Dashboard showing total Income, Expenses, and Savings
* Add and manage financial transactions
* Transaction history table with date, description, type, and amount
* Search and filter transactions by type
* Automatic balance calculation
* Interactive charts to visualize financial data
* Monthly financial report using charts
* Data storage using browser Local Storage

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js for data visualization
* Font Awesome for icons

## Project Structure

Personal_Finance_Dashboard

* index.html (Main dashboard page)
* transactions.html (Transaction management page)
* reports.html (Reports and charts page)
* style.css (Styling for the web application)
* script.js (Application logic and functionality)

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open the file **index.html** in any modern web browser.
4. Use the dashboard to add and manage transactions.
